# ESLint
    ESLint是一个代码规范检查工具，主要是用来做项目编码规范检查
    基本原理: 定义了很多规则, 检查项目的代码一旦发现违背了某个规则就输出相应的提示信息
    有相应的配置, 可定制检查
    
# ESlint主要提供以下支持：
    ES，JSX，style检查，自定义错误和提示
    .eslintrc.js是eslint忽略检查的文件，可以在里面的rule修改规则
